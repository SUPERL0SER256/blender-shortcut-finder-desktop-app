Blender Buddy-
The following is a small desktop app/ widget used to help look for Blender short cuts using the actions
Search up the action in the search bar and ull get the following shortcut.
Dependencies are axios and cheerio
The widget is an electron based application
HTML, CSS and Javascript was used to create the following application.
Once downloaded open in code editor of choice and run command 'npm run start' to open the app.
Hopefully it helps.
